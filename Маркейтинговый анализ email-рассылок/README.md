# Анализ эффективности email-рассылок

Проект по оценке эффективности 218 email-кампаний (24 признака).

## Данные

Используются агрегированные данные по рассылкам:
- отправлено / доставлено
- открытия и клики
- отписки и баунсы
- тема письма, сегмент, дата, время
- шаги воронки продаж

На основе исходных полей рассчитаны производные метрики:
- Delivery Rate
- Open Rate
- Click-to-Open Rate
- Unsubscribe Rate

## Задачи

- Очистка и приведение типов данных  
- Расчёт ключевых метрик и нормализация  
- Сравнительный анализ тем писем  
- Фильтрация кампаний по порогам метрик  
- Выявление топ-тем по эффективности и интересу контента  

## Подход

- Pandas + сводные таблицы и фильтрация  
- Два рейтинга:
  1. **Эффективность** — конверсии воронки, переходы на сайт, открываемость, низкие отписки  
  2. **Интересность** — переходы на сайт, открываемость, низкие отписки  
- Топовые темы выделяются по порогу рейтинга 0.8  

## Результаты

- **Лучшие темы:**  
  - Эффективность: **158**  
  - Интересность: **207**  
  - Совместная оценка: **158**  

## Вывод

Тема **158** — оптимальная по совокупной оценке: высокая эффективность воронки и привлекательность контента.
