# Анализ эффективности email-рассылок

Небольшой аналитический проект по оценке эффективности email-кампаний на основе выгрузки метрик рассылок (218 кампаний, 24 признака).

## Данные

Используются агрегированные данные по рассылкам:
- отправлено / доставлено
- открытия и клики
- отписки и баунсы
- тема письма, сегмент, дата, время
- шаги воронки продаж

На основе исходных полей рассчитаны производные метрики:
- Delivery Rate
- Open Rate
- Click-to-Open Rate
- Unsubscribe Rate

## Задачи

- приведение типов данных  
- расчёт ключевых метрик рассылок  
- сравнительный анализ тем писем  
- фильтрация кампаний по порогам метрик  
- выбор наиболее эффективной темы письма

## Подход

Использован pandas + сводные таблицы и фильтрация по метрикам.  
Отбор «лучших» кампаний выполнен по условиям:
- высокий Open Rate  
- высокий Click-to-Open Rate  
- низкий Unsubscribe Rate  

## Результат

Определены топ-темы писем по совокупности показателей.  
Лучший результат показала тема **194** — оптимальный баланс открываемости, низких отписок и большой массовости рассылки.
